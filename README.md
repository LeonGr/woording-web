woording-web
============
The Woording SPA, Work in progress.  
Built using VueJS  

## Features
- Browse your own and friends' lists
- Send friend requests
- Create lists
- No refreshes 
- configure list training
- Create account and log in system

## Not implemented yet
- Actually training lists
- Localization
- Beautifull landing page

## How to run
```bash
npm install
npm run dev
```

Don't.
